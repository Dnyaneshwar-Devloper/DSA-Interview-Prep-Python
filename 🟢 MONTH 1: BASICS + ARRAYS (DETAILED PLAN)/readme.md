## 📅 WEEK 1: Basics & Complexity Analysis
*Focus: Shifting from "it works" to "it's efficient."*

### 🧠 Topics
- **Big O Notation:** $O(1)$, $O(\log n)$, $O(n)$, $O(n \log n)$, $O(n^2)$.
- **Python Internals:** How `list` (dynamic array) and `dict` (hash map) work under the hood.
- **Dry Running:** Tracing code on paper to find logical errors.

## 📅 WEEK 2: Two Pointers Pattern
*Focus: Optimizing $O(n^2)$ nested loops into $O(n)$ linear passes.*

### 🧠 Topics
- **Opposite Ends:** Meeting in the middle (e.g., Two Sum II).
- **Slow & Fast Pointers:** Gap management (e.g., Remove element in-place).
- **Sorted Arrays:** Why sorting makes Two Pointers powerful.

### 🚀 Key Problems
1. **Two Sum II** (Input array is sorted)
2. **3Sum** (The classic Medium challenge)
3. **Container With Most Water**
4. **Valid Mountain Array**

---

## 📅 WEEK 3: Sliding Window Pattern
*Focus: Handling contiguous segments of data (Subarrays/Substrings).*

### 🧠 Topics
- **Fixed Window:** Constant size $K$ (e.g., Max sum of $K$ elements).
- **Variable Window:** Dynamic size based on conditions (e.g., Longest substring).
- **Hash Map Integration:** Using dictionaries to track characters in a window.

### 🚀 Key Problems
1. **Maximum Sum Subarray of Size K**
2. **Longest Substring Without Repeating Characters**
3. **Minimum Size Subarray Sum**
4. **Permutation in String**

---

## 📅 WEEK 4: Advanced Arrays & Strings
*Focus: Interview classics and edge-case handling.*

### 🧠 Topics
- **Kadane’s Algorithm:** Solving the Maximum Subarray problem in $O(n)$.
- **Boyer-Moore Voting:** Finding the Majority Element.
- **Matrix Manipulation:** 2D Arrays (Spiral Matrix, Rotate Image).

### 🚀 Key Problems
1. **Maximum Subarray (Kadane’s)**
2. **Majority Element**
3. **Merge Intervals**
4. **Product of Array Except Self**

## 🛠️ Study Rules (The "Golden" List)
1. **Don't Memorize:** If you can't explain it to a 5-year-old, you don't know it.
2. **20-Minute Rule:** If you're stuck for >20 mins, look at a hint. If >45 mins, look at the solution logic (not code).
3. **Hand-Write Logic:** Before typing a single line of Python, write the pseudocode.
4. **Pythonic Way:** Use `enumerate()`, `zip()`, and list comprehensions where applicable.
